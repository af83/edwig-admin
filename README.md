# Edwig_admin

Ruby 2.2.6

## Installation

### Application

```
git clone git@github.com:af83/Edwig-admin.git
cd Edwig-admin
```

### Postgresql

```psql
CREATE USER "edwig_admin" SUPERUSER PASSWORD 'edwig_admin';
CREATE DATABASE "edwig_admin";
```

Database configuration can be defined in config/database.yml
